主笔 / Claude Editorial
— 本周观察 / Weekly Narrative

这一周的 AI 行业，用一个词概括是"重装对峙"。三股力量同时亮牌：开源阵营以 DeepSeek-V4 的六分之一定价发起成本冲击；闭源阵营以 Google 400 亿投 Anthropic、Amazon 50 亿注资的规模筑起资本护城河；而 OpenAI 在周四投下 GPT-5.5，试图以 agentic 能力重新定义前沿。没有一方能单独定义这个时代，但每一方都在逼迫其他人加速。

与此同时，Agent 从概念走向基础设施的信号密集到无法忽视。OpenAI Workspace Agents、Google Cloud Next 的企业 Agent 平台、Microsoft Office Agent Mode、Salesforce Agentforce 2.0——一周之内四大平台同时发力 Agent 产品化。然而 Stanford 的研究冷静地指出，单 agent 往往不输多 agent，85% 的企业试点了 agent 却只有 5% 敢投产。热闹之下，信任架构的缺失才是真正的瓶颈。

第三条暗线是安全与失控的拉锯。Anthropic Mythos 模型泄露、Claude 性能下滑复盘、Meta MCI 追踪员工行为——这些事件串联起来，指向一个不舒服的现实：AI 公司在安全问题上的表现，远未达到它们自己宣称的标准。这条主线在下周的 Musk-Altman 庭审中还会继续发酵。
